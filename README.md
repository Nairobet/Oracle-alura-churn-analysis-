# Oracle-alura-churn-analysis-
oracle-alura-churn-analysis   Análisis predictivo churn telecom.  Este proyecto realiza un análisis predictivo integral del churn en telecomunicaciones, identificando drivers críticos de abandono (contrato mes +47% riesgo). Incluye modelado ML 85.2% precisión, visualizaciones estratégicas C-Level y recomendaciones ejecutivas para retención.
